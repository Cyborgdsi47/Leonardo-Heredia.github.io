SAFRASENSE

// MONITORAMENTO AGRÍCOLA POR SATÉLITE · SENTINEL-2 ESA

Plataforma web full-stack que democratiza o acesso a seguros agrícolas para pequenos e médios produtores rurais. O agricultor desenha o polígono de sua propriedade no mapa, cruza com o CAR (Cadastro Ambiental Rural) e recebe um dashboard com o índice NDVI calculado a partir de imagens reais do satélite Sentinel-2 da ESA. Gera laudos em PDF validáveis por Swiss Re e Mapfre.

// BACK-END
FastAPI · Python
  • Copernicus Data Space (ESA) via sentinelhub SDK — imagens Sentinel-2 L2A
  • Shapefile CAR (~3 GB) com GeoPandas + Shapely via bounding box antes do intersect
  • Cálculo NDVI: mín / med / máx por semana com NumPy
  • Série histórica de 52 semanas para detecção de estresse hídrico
// FRONT-END
React 19 · TypeScript
  • Dashboard NDVI: Saudável / Atenção / Emergência
  • Mapa interativo Leaflet para desenho do polígono da propriedade
  • Gráfico histórico semanal de NDVI dos últimos 12 meses (Recharts)
  • Laudo PDF para seguradoras (jsPDF) · Firebase Auth + Firestore
  • Multilíngue PT / ES / EN — Brasil, Paraguai, Bolívia

TUPÃ

// BASE DE REFERÊNCIA AMBIENTAL · CÓDIGO FLORESTAL

Plataforma web full-stack que automatiza a geração de bases de referência ambiental para imóveis rurais cadastrados no CAR (Cadastro Ambiental Rural). Cruza satélite Sentinel-2, hidrografia BHO/ANA e relevo SRTM/NASA com as regras do Código Florestal (Lei 12.651/2012) via PostGIS, entregando diagnósticos de conformidade ambiental em minutos.

// CAR
// POSTGIS
// CÓDIGO_FLORESTAL
// GEOPROCESSAMENTO
// BACK-END
FastAPI · PostgreSQL · PostGIS
  • Pipeline 5 etapas: Sentinel-2 → MapBiomas → Hidrografia → Declividade → Código Florestal
  • Geoprocessamento 100% em SQL — ST_Buffer, ST_Intersection, ST_Intersects via PostGIS
  • APPs calculadas por largura do rio (30 m a 200 m) e nascentes (raio 50 m)
  • Regras do Código Florestal em YAML — adaptável por estado ou país sem tocar no código
  • Score de conformidade (0–100) e exportação GeoJSON padrão OGC/SICAR
// FRONT-END
React 18 · TypeScript · Vite
  • Painel do Analista: seleção de CAR, disparo do pipeline e mapa das feições
  • Portal do Agricultor (/portal/imovel/:car) com notificações em tempo real
  • Mapa interativo Leaflet com camadas coloridas por tipo de feição ambiental
  • Sistema analista→agricultor com polling a cada 30 segundos
  • Auto-fallback para mock quando backend indisponível, com banner de aviso

// VER_EXPLICACAO_ANALISTA

Painel do analista ambiental do SFB. Mapa satélite com CAR delimitado em amarelo. Motor de análise com 100% de confiança via Sentinel-2 — gera automaticamente APP curso d'água, APP nascente e Reserva Legal. Diagnóstico identificou 18 divergências incluindo cultivo agrícola em faixa de APP. Botão "Enviar notificação" despacha alerta para o portal do agricultor com um clique.
